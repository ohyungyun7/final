1. 로그인
- 아이디 : 학번 및 사번 // 비밀번호 : 대소문자 구분없이 영문자, 숫자, 특수문자를 조합한 8~12자리
- 로그인 3회 실패시 캡차인증(API)를 해야한다.
- 아이디 찾기 : 아이디 찾기 모달창이 뜬다.
	        이름, 핸드폰번호, 이메일을 입력받는다.
                위 3가지와 일치하는 정보가 조회될 경우 아이디를 모달창에 띄운다.
- 최초 비밀번호 : 초기 비밀번호는 회원의 생년월일 6자리로 자동 부여된다.
                  최초 비밀번호로 로그인 시 비밀번호 변경하라는 알림창이 뜨고, 비밀번호 변경 모달창이 뜬다.
- 비밀번호 찾기 : 비밀번호는 대소문자 구분없이 영문자, 숫자, 특수문자를 조합한 8~12자리로만 변경이 가능하다.
                  최초비밀번호로 입력한 후의 변경이 아닌 로그인 화면에서 비밀번호 찾기를 클릭하면
                  우선 아이디와 핸드폰 번호를 입력받고, 이 정보에 일치하는 회원이 있을 경우,
                  구글 OTP API를 이용하여 2차 본인인증을 진행한다.
                  비밀번호는 변경할 비밀번호와 변경할 비밀번호 확인을 위해 2번을 입력해야하며, 일치할 시 비밀번호가 변경된다.
                  이후 모달창을 끄고 로그인을 진행하면 된다.

* 캡챠인증 API : 
* 구글 OTP API : 회원이 구글 OTP 어플을 다운받아 홈페이지에 출력된 번호를 어플에 입력하고, 출력된 번호와 입력된 번호가 일치할 시
                     본인인증을 완료할 수 있다. 		


2. 음성인식
- 메인페이지 : 로그인 이후 메인페이지에서 음성검색 버튼을 클릭할 시 음성인식 API를 활용할 수 있다.
                  이때 음성언어를 텍스트로 변환하여 메인페이지에서 다른 페이지로 이동할 수 있다.
                  (ex. 아이콘 클릭 후 '마이페이지'라고 말했을 때 마이페이지로 이동한다.)
- 쪽지함 : 쪽지함에서 검색바에서 음성검색 버튼을 클릭할 시 음성인식 API를 활용할 수 있다.
            이때 쪽지전송자, 쪽지내용 등과 관련된 음성언어를 텍스트로 변환하여 검색할 수 있다.
+ 이외에 조회, 검색과 관련된 페이지에서 음성인식 API를 사용할 수 있고, 위와 같은 방식으로 조회 및 검색이 가능하다.


3. 학사일정 (일정조회 / 일정관리)
- 조회 (학생, 교수는 조회만 가능하다.)
  학사일정은 연도별로 조회 가능하다.
  처음 조회화면는 금년도 일정이 자동으로 화면에 출력된다. 금년도 이외의 일정을 조회하기 위한 select태그가 존재한다.
  메인섹션 상단은 1월부터 12월까지 조회할 수 있는 버튼이 나열되어 있고,
  메인섹션 중단 왼쪽부분은 캘린더가 있으며, 오른쪽 부분에는 해당 월의 일정을 간략하게 확인할 수 있다. 
  메인섹션 하단은 캘린더에 작성된 일정 클릭시, 자세한 내용을 확인할 수 있다.

(월별 조회 방법 1. 캘린더에 '<', '>'버튼을 클릭하여 월을 이동할 수 있다. 동시에 12월에서 1월 이동, 1월에서 12월 이동시 년도 이동이 가능하다.
                    2. 연도별로 모든 월이 한 페이지에 출력되며, 스크롤을 이용하여 월을 이동할 수 있다.)

- 관리(교직원은 조회 및 수정이 가능하다.)
  조회는 위와 동일하다.
  수정화면은 조회화면과 동일하다.
  일정 추가, 수정, 삭제가 가능하다.
  월별 캘린더의 일자를 클릭하여 모달창을 통해 일정 변경이 가능하다.
  일정의 시간, 기간, 일정키워드(캘린더 표시용), 간략한 일정내용(메인섹션 중단 오른쪽부분 표시용), 상세한 일정내용(메인섹션 하단 표시용)을
  입력할 수 있다.

(중요일정이 추가, 변경, 삭제가 되었을 시 쪽지를 보내거나 공지사항을 작성하는 것이 시스템적으로 나을지 고민됨.)

4. 마이페이지 (개인정보 확인 및 수정)
   마이페이지의 기본적인 기능은 모든 사용자에게 동일하게 적용된다.
   마이페이지에서 개인정보 확인 및 수정이 동시에 가능하다.
   (바로 수정이 가능하기 때문에 로그인 비밀번호와 동일한 비밀번호를 입력해야 이동이 가능하다.) 
   
- 학생 
  확인 내용 : 단과대학명, 학과명, 학번, 학년, 이름,
 생년월일, 국적 등은 조회만 가능하며 수정할 수 없다.
  수정 내용 : 사진, 비밀번호, 이메일, 영문이름, 병역, 장애유무,
                 거주지, 핸드폰번호, 개인정보 제공동의 체크, 계좌번호 등을 수정할 수 있다.

- 교수
  확인 내용 : 단과대학명, 학과, 직급, 사번, 이름, 생년월일, 국적 등은 조회만 가능하며 수정할 수 없다.
  수정 내용 : 사진, 비밀번호, 이메일, 영문이름, 병역, 장애유무, 거주지, 핸드폰번호, 개인정보 제공동의 체크, 계좌번호,
                 학력, 수상이력, 활동이력 등을 수정할 수 있다.

- 교직원
  확인 내용 : 소속명, 직급, 사번, 이름, 생년월일, 국적, 근무기간 등은 조회만 가능하며 수정할 수 없다.
  수정 내용 : 사진, 비밀번호, 이메일, 영문이름, 병역, 장애유무, 거주지, 핸드폰번호, 개인정보 제공동의 체크, 계좌번호 등을 수정할 수 있다.


5. 교직원 조회 (교직원 조회 및 검색 / 교직원 정보 페이지 / 교직원,교수 등록 / 교수평가 조회)
- 일반회원(학생, 교수, 인사팀 외의 교직원)
  소속명, 이름, 교내전화번호 등의 기본적인 사항만 조회 및 검색이 가능하다.
  교직원/교수 조회창에서 소속명 선택시 그에 해당되는 직원과 직원의 일반사항을 테이블 식으로 조회할 수 있다.
  조회된 교직원에게 쪽지를 보낼 수 있다.
 
- 인사팀 교직원
  기본적인 사항 외의 자세한 내용을 조회 및 검색할 수 있다.
  소속명 선택시 그에 해당되는 직원과 직원의 일반사항을 테이블 식으로 조회할 수 있고, 직원명을 클릭 시 교직원 상세정보를 확인할 수 있다.
  상세정보페이지는 마이페이지의 개인정보 확인 페이지와 동일하다.
  기본적으로 조회만 가능하며, 수정해야할 시 인사팀의 고유 비밀번호를 입력해야 한다.

- 교직원 등록
  학생 등록과 동일하게 엑셀 API를 이용하여 한번에 교직원과 교수를 등록할 수 있다.

- 교수평가 조회
  : 교수평가 항목 관리
    전공수업, 필수교양수업, 일반교양수업별로
    대면수업, 비대면수업별로 평가 항목을 관리할 수 있다.
    분류별로 평가 항목이 동일하다.
    항목별로 5가지 단계별 선택할 수 있게 한다.

- 교수평가 결과 확인
  : 단과대학, 과목단위별로 평가 결과를 조회할 수 있다. 
    교수평가의 결과를 항목별 선택지에 따라 점수화할 수 있으며, 평균점수에 따라 상중하로 교수들 항목을 나눌 수 있다.
